# Mental Health Support System 🧠💙

A comprehensive Digital Psychological Intervention System designed specifically for college students, providing stigma-free mental health support through AI-guided assistance, confidential counselling, and peer support networks.

## 🌟 Features

### 1. AI-Guided First-Aid Support
- **Intelligent Chat System**: 24/7 AI mental health assistant with crisis detection
- **Crisis Intervention**: Automatic detection of crisis situations with immediate professional referral
- **Coping Strategies**: Personalized coping techniques and stress management advice
- **Confidential Support**: Anonymous conversations with no data retention

### 2. Confidential Booking System
- **Professional Counsellors**: Access to qualified campus mental health professionals
- **Flexible Scheduling**: Easy appointment booking with real-time availability
- **Urgency Levels**: Priority-based scheduling for urgent mental health needs
- **Secure Platform**: End-to-end encrypted communication and data protection

### 3. Psychoeducational Resource Hub
- **Multi-format Content**: Videos, audio guides, documents, and interactive resources
- **Multi-language Support**: Resources available in English, Hindi, Spanish, Mandarin, and French
- **Categorized Content**: Organized by mental health topics and concerns
- **Offline Access**: Downloadable resources for offline use

### 4. Peer Support Platform
- **Trained Volunteers**: Verified peer support volunteers with specialized training
- **Moderated Environment**: Safe, supportive community with professional oversight
- **Anonymous Support**: Confidential peer-to-peer assistance
- **Live Chat**: Real-time connection with peer volunteers

### 5. Mental Health Assessment Tools
- **PHQ-9 Screening**: Standardized depression assessment tool
- **GAD-7 Screening**: Anxiety disorder screening questionnaire
- **Personalized Recommendations**: Actionable insights based on assessment results
- **Progress Tracking**: Monitor mental health improvements over time

### 6. Admin Dashboard
- **Anonymous Analytics**: Data-driven insights without compromising privacy
- **Trend Analysis**: Identify mental health patterns and intervention opportunities
- **Resource Allocation**: Optimize mental health service distribution
- **Crisis Monitoring**: Track and respond to mental health emergencies

## 🚀 Technology Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **Charts**: Recharts for data visualization
- **Icons**: Lucide React
- **State Management**: Zustand
- **Forms**: React Hook Form

## 📋 Prerequisites

- Node.js 18+ 
- npm or yarn package manager
- Modern web browser

## 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd mental-health-support-system
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
